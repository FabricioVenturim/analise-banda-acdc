Como usar
=========

*Passo 1*: utilize o comando a seguir para instalar as bibliotecas necessárias 
para o funcionamento do programa:

.. code-block:: console

    pip install -r requirements.txt

*Passo 2*: Para executar o programa, é necessário rodar o arquivo main.py da
pasta. Nele há apenas um código que chamará a função interface do módulo.

*OBS*: Certifique-se de que os dois csv disponibilizados estão pasta main
